Requirements:

termcolor==1.1.0
python==3.6.7

Installation:

pip3 install termcolor

Controls:

W -> UP  
S -> DOWN  
A -> LEFT  
D -> RIGHT  
X -> DROP A BOMB  
Q -> QUIT GAME

How to Run The Game :

Run python3 game.py

Note : The game is independent of the rate of input, and the player moves once in a second
